# AI-mock-interview-
AI 模拟面试
基于deepseek LLM 大模型+百度语音识别+语音转文字生成AI 智能面试助手。 支持实时多轮与虚拟面试官对话，深度追问，还原真实面试场景。 
功能亮点：
真实对话感，
针对不同岗位的海量题库
面试追问开始压力模型。

This is an AI interview assistant powered by Deepseek LLM, Baidu speech recognition, and the text-to-speech technique. It supports real-time voice conversation and in-depth follow-up questioning./
